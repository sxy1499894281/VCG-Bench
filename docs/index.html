Benchmark

From pixels to editable diagram code.

VCG-Bench evaluates whether multimodal models can recover structured Draw.io / mxGraph XML from diagram images, and then edit that XML under natural-language instructions.

1

Vision-to-XML generation

Models generate executable mxGraph XML that must parse, render, and preserve the original diagram's layout, text, topology, and style.

2

Instruction-based XML editing

Models predict deterministic XML patches for precise diagram edits, separating visual understanding from code-to-code modification ability.

3

Executable evaluation

Metrics cover renderability, structure, visual style, semantic consistency, and edit preservation instead of relying on text matching alone.

A synthetic data engine for model cold-start.

The released Task 1 dataset provides diagram images, domain labels, structured descriptions, and restored mxGraph XML. The pipeline is designed to create verified visual-code pairs that can bootstrap open-source diagram-to-code models.

What the release contains.

The project separates benchmark construction, runnable code, released data, and agent-assisted reconstruction workflows.

Code

VCG-Bench

Generation scripts, evaluation pipelines, prompt templates, examples, and local model helpers for Draw.io / mxGraph workflows.

Data

Hugging Face Dataset

Task 1 parquet release with 1,449 samples across academic, business, software, management, general, and UI/UX domains.

Skill

Draw.io Reconstruction

A Codex skill for high-fidelity diagram reconstruction into editable Draw.io files with visual audits and multi-turn refinement.

High-fidelity reconstruction with Codex.

VCG-Bench provides data and evaluation. For production-quality reconstruction of complex diagrams, the companion drawio-reconstruction skill supports agent-in-the-loop editing: inspect the reference, rebuild editable structure, export a preview, and refine defects through conversation.
