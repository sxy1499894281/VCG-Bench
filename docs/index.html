Synthetic data and evaluation for visual-centric structured diagram generation and editing.

VCG-Bench turns professional diagrams into executable, editable Draw.io / mxGraph XML tasks, enabling cold-start supervision for open-source models and precise evaluation beyond pixels.

Benchmark

From raster diagrams to editable code.

VCG-Bench evaluates models in the diagram-as-code setting: outputs must be valid, renderable, visually faithful, and practical to edit.

Vision-to-XML generation

Recover executable Draw.io / mxGraph XML from professional diagram images.

Instruction-based editing

Apply natural-language edits through deterministic XML patch prediction.

Fine-grained evaluation

Measure execution success, visual style, semantic fidelity, and edit preservation.

Synthetic Data

A cold-start engine for open-source diagram models.

The data pipeline combines machine generation and validation: collect diagrams, synthesize descriptions and XML, render outputs, filter by parseability and visual consistency, then release verified visual-code pairs.

01Collect

Gather diverse diagrams from professional domains.

02Synthesize

Generate structured captions and candidate mxGraph XML.

03Validate

Render and filter by executable structure and visual consistency.

04Release

Provide training-ready data and evaluation protocols.

Reconstruction Skill
